- `gitlab-ci-guide.md` - GitLab CI specific documentation

- `pipeline-reference.md` - Complete reference for all pipeline configurations

## Contributing to Documentation

When adding new pipelines or templates, please:

2. Add examples and usage instructions
3. Include troubleshooting information if applicable
4. Follow the established documentation format
